# Decision-Tree-Random-Forest
Proyek ini menganalisis dataset Kebahagiaan 2019 untuk mengklasifikasikan negara ke dalam kategori kebahagiaan tinggi/rendah. Menggunakan Decision Tree dan Random Forest, kami membandingkan kinerja model. Random Forest unggul dalam prediksi, dan analisis feature importance mengungkap faktor kunci kebahagiaan seperti GDP dan dukungan sosial.
